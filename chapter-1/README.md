# Chapter 1
Start learning how to compile in rust with the compiler `rustc` and the basic `cargo` commands.

### Important cargo commands
- `cargo new [project_name]`: create a new project with `cargo`
- `cargo build`: create a not optimized build for development
- `cargo run`: run the compiled code
- `cargo check`: check if the code has compiler errors
- `cargo build --release`: optimized build for the release of a crate
